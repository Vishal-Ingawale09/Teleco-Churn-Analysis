# Teleco Customer Churn Analysis

## Project Overview
This project analyzes customer churn in a telecom company using Python.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Objectives
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Customer Churn Analysis
- Data Visualization

## Author
Vishal Ingawale

## 📂 Dataset
The project uses a Telecom Customer Churn dataset containing customer demographic information, account details, subscribed services, and churn status.

## 📊 Key Insights
- Approximately 26.5% of customers have churned.
- Customers with month-to-month contracts show a higher churn rate.
- Customers with shorter tenure are more likely to churn.
- Customers without Online Security and Tech Support services have a higher probability of churning.
- Senior citizens have a relatively higher churn rate compared to other customer groups.
